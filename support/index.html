How do I set up the keyboard?

Settings → General → Keyboard → Keyboards → Add New Keyboard → Cipher. To use translate-as-you-type, also turn on Allow Full Access for Cipher (this is what lets the keyboard reach the network for translation — it’s the only thing it’s used for).

What is Cloud Translate, and is it private?

It’s the keyboard’s optional translate-as-you-type. It’s off until you enable it. When on, only the text you’re translating is sent over an encrypted connection to a translation service and returned — no account, no identifiers, nothing stored. It’s automatically disabled in password and other sensitive fields. Everything else in the app (voice, camera, text) translates entirely on your device with nothing uploaded.

How do I cancel my subscription?

Settings → Apple Account → Subscriptions → Cipher → Cancel. The 7-day trial won’t charge you if you cancel before it ends.

How do I restore a purchase?

On the Pro/paywall screen, tap Restore Purchases.

Which languages are supported?

19 languages on-device, including Spanish, French, German, Italian, Portuguese, Dutch, Polish, Turkish, Ukrainian, Russian, Arabic, Hindi, Indonesian, Vietnamese, Thai, Chinese, Japanese, Korean, and English.
